Free learning resources for the beginner level developer.

Post on 20 Jan 2023 | 4 min read

Today I am going to share a free learning resource list for the developers that you can learn about different Programming languages and Technologies. This is also my favorite list as given below.

1. w3 School

W3Schools is a school for web developers, covering all the aspects of web development: HTML Tutorial. CSS Tutorial. JavaScript Tutorial.

2. Javatpoint

JavaTpoint provides easy and point to point learning of various online tutorials such as Java, Android, Design Pattern, JavaScript, AJAX, SQL, Artificial Intellegence, Cloud Computing, Hadoop, Data Science, Machine Learning, Blockchain, Angular, and Python etc.

3. Programiz

Programiz is a platform to learn programming built by programmers for programmers. With more than 50 million enthusiasts relying on Programiz to learn programming, we pride ourselves in being among the leaders in the industry.

4. Geeksforgeeks

GeeksforGeeks is an portal for computer geeks where you will find a bunch of articles, quizzes on programming and algorithm. You will find problems as categorized in school, easy, medium, hard,advanced.

5. MDN Web Docs

MDN Web Docs (formerly known as the Mozilla Developer Network or MDN) is a free resource for in-depth documentation on web standards such as HTML5, CSS, JavaScript, and much more. MDN's mission is simple: provide developers with the information they need to easily build projects on the open Web.

6. Udemy

This is learning platform where you can find all types of courses. On this platform courted are teaching by different educators. This paid platform you can course at nominal prices. Sometimes educators & teachers offers free course also.

7. GreatLearning

This is a platform where we can take different technology courses. This platform supports both free and paid courses do check.



In my opinion this is best resources/platform to learn programing and technology and you can also find much more resource which available on the internet.

Thanks for the reading and hope you find it useful!
